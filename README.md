# Hyperledger Caliper ベンチマーク

このリポジトリには、ブロックチェーンパフォーマンスベンチマークフレームワークであるhyperledger Caliperで使用できるサンプルベンチマークが含まれています。
[Caliperメインリポジトリ](https://github.com/hyperledger/caliper/)をフォークし、信用スコアに基づく独自のNFTのスマートコントラクトのベンチマークを行っています。

これらのベンチマークの実行に基づくパフォーマンスレポートは、[リポジトリのGitHub Pages](https://hyperledger.github.io/caliper-benchmarks/)に公開されています。

## mainブランチの内容
mainブランチに含まれるベンチマークは、3つのディレクトリに分かれています：

1. **benchmarks**. テスト設定とコールバックファイルで構成されています。テスト設定ファイルは、ベンチマークテストのパラメータを記述し、ベンチマーク中にCaliperクライアントによって実行されるコールバックファイルを参照します。Benchmarkフォルダには以下のサブフォルダが含まれます：
    - **api** 単一のターゲットブロックチェーンのAPIに向けたテスト。
	- **samples** ターゲットブロックチェーンプラットフォームが提供するネイティブサンプルに向けたテスト。
	- **scenario** すべての（サポートされている）ターゲットブロックチェーンプラットフォームに有効な汎用シナリオ
		- **scoring** 信用スコアを活用した認可システムを組み合わせたERC-721のスマートコントラクトを発行・転送する
2. **networks**. ベンチマーク目的でテスト対象システムとして使用できるサンプルブロックチェーンネットワークで構成されています。
3. **src**. SUTにデプロイされ、benchmarksフォルダ内のテストコールバックを介して対話されるソーススマートコントラクトファイルで構成されています。各スマートコントラクトは、対応するブロックチェーン技術の下にある独自のフォルダに保持されています。

## ベンチマークの実行
このリポジトリに含まれるベンチマークを実行するには、すべてのファイルの使用を想定している[Hyperledger Caliper](https://github.com/hyperledger/caliper/)をインストールする必要があります。

GoQuorumネットワークでこのリポジトリを使用する詳細については、[GoQuorumネットワーク](/networks/quorum/4node/README.md)をご覧ください
